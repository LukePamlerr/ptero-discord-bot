Pterodactyl Panel Deployment

1

Check for Python Egg

First, check if you already have the Python egg installed in your Pterodactyl panel.

  • Go to Admin ‚Üí Nests
  • Look for "Python" or "Generic Python" nest
  • If you see it, you can skip to Step 3
2

Install Python Egg

If you don't have the Python egg, install it from the official repository.

  1. Download Python egg from: Official Repository
  2. Go to Admin ‚Üí Nests ‚Üí Import Egg
  3. Upload the downloaded egg-python-generic.json file
  4. This will create a "Python" nest with the generic Python egg
3

Create Discord Bot Server

Create a new server for your Discord bot using the Python egg.

  • Navigate to Servers ‚Üí Create New Server
  • Select "Python" nest and "Generic Python" egg
  • Server Settings:
    • Name: Pterodactyl Discord Bot
    • Memory: 1024MB (minimum), 2048MB (recommended)
    • CPU: 1 core (minimum), 2 cores (recommended)
    • Disk: 2GB (minimum), 4GB (recommended)
    • Database: Create a new PostgreSQL database
4

Configure Environment Variables

Set up the required environment variables for your bot.

Variable Example Required
DISCORD_BOT_TOKEN abcd1234... ‚úÖ
DATABASE_URL postgresql://user:pass@localhost:5432/db ‚úÖ
DISCORD_GUILD_ID 123456789012345678 ‚ùå
BOT_PREFIX ! ‚ùå
LOG_LEVEL INFO ‚ùå
5

Upload Bot Files

Upload the required bot files to your server.

  • Download the bot source code from GitHub
  • Required files to upload:
    • bot.py (main entry point)
    • requirements.txt (Python dependencies)
    • bot/ directory (all bot modules)
    • alembic.ini (database migrations)
    • alembic/ directory (migration files)
  • Upload these files through the Pterodactyl file manager or SFTP
6

Start the Bot

Launch your Discord bot and verify it's working correctly.

  1. Click Start Server
  2. Monitor the console for startup logs
  3. The bot will automatically connect to Discord and register commands
  4. Check your Discord server to see the bot online
Your Discord bot is now running on Pterodactyl!

Discord Bot Setup

Create Discord Application

  1. Go to Discord Developer Portal
  2. Create a new application
  3. Add a bot to the application
  4. Enable Server Members Intent and Message Content Intent
  5. Copy the Bot Token

Invite Bot to Server

  1. Generate OAuth2 URL with permissions:
    • applications.commands
    • bot
    • administrator (recommended)
  2. Invite the bot to your Discord server
  3. Verify the bot appears in your server member list

Database Setup

  1. Use Pterodactyl's built-in database
  2. Database will be automatically created
  3. Use the connection string from server settings
  4. Format: postgresql://pterodactyl:password@localhost:5432/database_name
  5. Run migrations: python -m alembic upgrade head

Troubleshooting

Bot Won't Start

  • Check environment variables are set correctly
  • Verify Discord bot token is valid
  • Check database connection string
  • Review console logs for errors

Commands Not Working

  • Ensure bot has proper Discord permissions
  • Check if commands are registered (wait 1-2 minutes after startup)
  • Verify user has configured their Pterodactyl panel
  • Check Discord API status

Database Connection Failed

  • Check database is running and accessible
  • Verify connection string format
  • Ensure database user has proper permissions
  • Run database migrations

Performance Issues

  • Increase server memory allocation
  • Check resource usage in Pterodactyl
  • Enable debug logging to identify bottlenecks
  • Consider upgrading server resources

Debug Mode

LOG_LEVEL=DEBUG

Enable debug logging through environment variable to get detailed error information.
